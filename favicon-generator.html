Favicon Generator Instructions

To replace the 'G' favicon with your new 'F' favicon, follow these steps:

  1. I've created an SVG file f-favicon.svg in your favicon directory with the letter 'F'
  2. Visit https://realfavicongenerator.net/
  3. Click on "Select your favicon image" and upload the f-favicon.svg file
  4. Adjust settings if needed (the defaults should work well)
  5. Click "Generate your Favicons and HTML code"
  6. Download the generated favicon package
  7. Extract the package and replace all files in /Users/federico/Desktop/fedeotto.github.io/assets/favicon/ with the new files
  8. Keep the existing browserconfig.xml and site.webmanifest files, but update them with any new settings from the generated ones
  9. Commit and push your changes
  10. Clear your browser cache and reload your website

Alternative method:

  1. Use https://favicon.io/favicon-generator/
  2. Select "Text" as the generator type
  3. Enter "F" as the text
  4. Choose a background color (#0056b3 for light theme or #64b5ff for dark theme)
  5. Choose white (#ffffff) for the text color
  6. Adjust other settings as desired
  7. Download the generated package and replace the files as described above
